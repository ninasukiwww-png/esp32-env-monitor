# ESP32 环境监测仪 - 备份参考点

> 备份时间: 2026-05-02
> 来源: ESP32 真机 (COM4) - MicroPython v1.27.0
> 已验证: 此版本已在真实硬件上运行通过

## 文件说明

| 文件 | 说明 |
|------|------|
| `main.py` | 核心程序 - 七段数码管显示温湿度, SD卡日志, WiFi联网, NTP校时 |
| `ili9341.py` | ILI9341 TFT 驱动 (240x320) |
| `sdcard.py` | SD 卡 SPI 驱动 |
| `boot.py` | ESP32 启动脚本 |

> `unused/` 文件夹存放了 ESP32 上的其他文件，不参与本系统运行
